HMM_Impl
========

对HMM模型的一些简单实现
HMM主要着力解决三大问题：

1，预测
2，解码
3，学习

预测是指给定HMM模型和观察序列，计算在给定模型的情况下得到观察序列的概率，这个是比较常见，也比较简单的。
通常比较有效的是采用前向算法

解码是指给定HMM模型和观察序列，计算要得到观察序列，最有可能的隐藏状态序列是什么。
这里通常采用Viterbi算法。

学习是指不给定HMM模型，根据给定的观察序列和隐藏状态，学习得到最优的HMM模型。
这一类问题比较复杂，通常采用前向-后向算法。
